## Meeting Room Bookingについて
<p>
小規模な企業で使用される想定の会議室予約アプリです。 <br />
<br />
サイドメニューの各フロアごとの会議室一覧から会議室を選択し、表示されるカレンダーをドラッグして予約します。　<br />
予約確定ボタンを押下でDjangoへ情報を送付しています。 予約の削除はカレンダーから該当予約をクリックすることで可能です。カレンダーの表示にはFullCalenderを使用しています。 <br />
</p>
